# HuaxiaTCM

## Huaxia TCM Heritage & Education Center

Traditional Chinese Medicine learning platform focused on preservation, education, and global promotion for global learners.

### Mission and Scope

- Preservation, education, and global promotion
- Supporting Sustainable Development Goals
- For global learners

### Six Key Modules

- Comprehensive encyclopedia
- Prescription Matching Game
- AI-powered TCM Assistant
- Quiz Module
- Interactive Story Mode
- Management Panel

### Alignment with SDGs

- SDG 3 (Good Health and Well-being)
- SDG 4 (Quality Education)

## Tech Stack

- Backend: Python (Flask), Gunicorn
- Frontend: Vue 3, Vite
- State/Router/I18n: Pinia, Vue Router, Vue I18n
- Mapping/GIS: Leaflet, Vue Leaflet, Turf.js, polyline
- UI/Data: Heroicons, vue-chartjs
- HTTP: Axios
- Document: docx
- Database: SQLite
